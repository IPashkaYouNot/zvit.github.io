Сценарій на JavaScript включається в документ за допомогою тега script

Розміщення сценарію в розділ head документа призводить до того, що сценарій буде завантажений до того, як буде потрібно його виконати.

Код сценарію розміщують в теги коментаря html для того, щоб старі браузери, які не розуміють JavaScript, не відображали цей код на екрані.

Регістр, в якому написані літери, в JavaScript має значення.

Файли з текстами програм на JavaScript мають розширення js.

Елемент script може розташовуватися як всередині секції head, так і всередині body. Web-документ часто містить кілька сценаріїв JavaScript. Тег SCRIPT, введений в елементі HEAD документа HTML завжди виконується першим. Хоча цей сценарій і не дозволяє відображати на екрані дані, він чудово підходить для визначення функцій.

Сценарій, який розміщений у тілі документа HTML, виконується завжди після сценаріїв, введених в його заголовку. Кілька сценаріїв в тілі документа HTML виконуються в порядку їх слідування.
